## Razumevanje
Mapa razumevanje vsebuje osnove modela. Tukaj se nahaja datoteko *osnovne_funkcije.R*, ki vsebuje vzpostavitev rojstno smrtnih procesov. Medtem je datoteka *raznoliko.R* namenjena bolj naprednim testom, ki govorijo o lastnostih procesov, poleg tega pa sta vzpostavljena tudi dva posebna procesa, in sicer čisto smrtni in čisto rojstni proces.
